Project objective:

To build, train and deploy a model on AWS which predicts whether the customer is going to churn in the near future or not.
Description:
Data Description: The dataset contains information about 10000 banking customers with various attributes such as CustomerId, Surname, CreditScore, Geography, Gender, Age, Tenure, Balance, NumOfProducts, HasCrCard, IsActiveMember, EstimatedSalary and Exited etc.
Tech Stack:
Language: Python
Libraries: pandas, numpy, scipy, matplotlib, seaborn, sklearn, Flask, Docker, AWS, Ensemble, Gunicorn, Terraform
Approach followed:
    1. Data Preprocessing:
  • Categorical Data Cleaning
  • Continuous Data Cleaning
  • Univariate Data Analysis
  • Multivariate Data Analysis
  • Outlier Treatment
  • Feature Extraction
  • Features Scaling
    2. Machine Learning Model Building:
  • Logistic Regression
  • Decision Tree Classification
  • Random Forest Classification
  • Gradient Boosting & Adaptive Boosting
  • Extreme Gradient Boosting Classification
  • Voting Classification
  • LightGBM Classifier
    3. Model Deployment:
  • APIs - Application Programming Interface
